# DjangoExp

A Django CRUD project with login authentication.

## Setup

1. Clone the repo
   git clone https://github.com/sahil-raj2712/DjangoExp.git

2. Create virtual environment
   python -m venv .venv
   .venv\Scripts\activate

3. Install dependencies
   pip install -r requirements.txt

4. Run migrations
   python manage.py migrate

5. Create superuser
   python manage.py createsuperuser

6. Start server
   python manage.py runserver



## Features
- Login / Logout
- Add student
- Edit student
- Delete student
- SQLite database
